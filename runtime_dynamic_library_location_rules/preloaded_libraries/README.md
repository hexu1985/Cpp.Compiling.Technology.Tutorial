### 预加载库

在库搜索过程中，绝对最高优先级始终保留给预加载指定的库——加载器会首先载入这些库，随后才开始搜索其他库。

指定预加载库的方式有两种：

- 通过设置 LD_PRELOAD 环境变量

```bash
export LD_PRELOAD=/home/milan/project/libs/libmilan.so:$LD_PRELOAD
```

- 通过 /etc/ld.so.preload 配置文件

该文件包含以空格分隔的ELF共享库列表，这些库将在程序启动前优先加载。

指定预加载库并非标准设计规范，而是用于特殊场景，比如设计压力测试、诊断分析以及原始代码的紧急修补。

在诊断场景中，您可以快速创建标准函数的定制版本，加入调试输出信息，并将其编译成共享库。
通过预加载该库，就能有效替换原本提供该函数的标准动态库。
